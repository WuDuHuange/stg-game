# STG机娘游戏

一款机娘风格的弹幕射击游戏（STG），具有丰富的动效和深度的游戏机制。

## 项目简介

STG机娘是一款融合了机娘风格与深度装备系统的弹幕射击游戏。游戏以丰富的动效、多样化的武装组合和策略性战斗为核心特色，为玩家提供爽快的射击体验和深度的自定义玩法。

## 核心特色

- **自由武装搭配**：5个槽位无兼容性限制，玩家可自由组合不同武装
- **技能装备绑定**：装备决定可用技能，通过击杀强化技能效果
- **多阶段升级系统**：击杀强化上限5级，属性自动提升
- **多样化敌人设计**：包含稀有度分级、弱点系统和多阶段Boss
- **动态关卡生成**：除Boss外随机生成关卡，二周目开启难度选择

## 技术栈

- **游戏引擎**: Phaser 3
- **开发语言**: TypeScript
- **构建工具**: Vite
- **包管理器**: npm
- **测试框架**: Vitest
